# 🚀 Microsoft Office 365 Auto-Installer
**Developed by ZERO STUDIO**

A streamlined, automated tool to install the official Microsoft Office 365 suite directly from Microsoft's servers using the Office Deployment Tool (ODT).

Skip the manual downloads and complex command lines. You can trigger the entire installation with a single command in **PowerShell**.

1. Click the **Start** menu.
2. Search for **PowerShell**.
